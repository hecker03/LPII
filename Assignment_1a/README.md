# 📊 Exam Admin Dashboard

A responsive and visually appealing **Exam Administration Dashboard** built using **HTML, CSS, and Bootstrap 5**.  
This project showcases a modern UI with a sidebar, colorful statistic cards, and an exam schedule table.

---

## 🚀 Features

- 📱 Fully Responsive Design (Mobile + Desktop)
- 🎨 Colorful Gradient Cards
- 🖱️ Smooth Hover Effects
- 📊 Dashboard Overview with Key Statistics
- 📅 Upcoming Exams Table

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5

---

## 📸 Dashboard Sections

### 🔹 Sidebar Navigation

- Dashboard
- Exams
- Students
- Results
- Schedule
- Settings

### 🔹 Statistics Cards

- Total Students
- Total Exams
- Ongoing Exams
- Results Published

### 🔹 Upcoming Exams Table

- Exam Name
- Date
- Department
- Status

---

## 🎯 Purpose of Project

This project is developed as part of a **Web Application Development (WAD) Lab Assignment** to demonstrate:

- Responsive Web Design using Bootstrap
- UI/UX Design Principles
- Dashboard Layout Implementation

---

## 💡 Key Concepts Used

- Bootstrap Grid System (`row`, `col-md`, `col-sm`)
- CSS Styling and Layout Design
- CSS Gradients
- Hover Effects using `transform` and `box-shadow`
- Responsive Design using Media Queries
