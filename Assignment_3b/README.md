# WAD Assignment - CRUD API (Node.js + Express + MongoDB)

## 📌 Objective

Create four APIs using Node.js, Express.js, and MongoDB to perform CRUD operations.

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose

---

## ⚙️ Features

- Create data (POST API)
- Read data (GET API)
- Update data (PUT API)
- Delete data (DELETE API)
- RESTful API structure
