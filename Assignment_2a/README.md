# WAD Assignment - GitHub Version Control

## 📌 Objective

Create a GitHub account and use Git commands to create a repository and push code.

---

## 🛠️ Tools Used

- Git
- GitHub

---

## ⚙️ Steps Performed

1. Created a GitHub account
2. Created a new repository
3. Initialized Git locally
4. Added project files
5. Committed changes
6. Connected local repo to GitHub
7. Pushed code to GitHub

---

## 💻 Git Commands Used

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin <repository-url>
git branch -M main
git push -u origin main
```
