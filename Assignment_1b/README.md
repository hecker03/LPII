# WAD Assignment - User Registration (AJAX + Local Storage)

## 📌 Objective

Create a JavaScript program to:

- Collect user registration data
- Send data using AJAX (POST method)
- Store data in Array / Local Storage
- Display data on a new page

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- AJAX

---

## ⚙️ Features

- User Registration Form
- AJAX POST request
- Data stored in Local Storage
- Display data in a new page

---

---

## ▶️ How to Run

1. Open `index.html`
2. Fill the form and submit
3. Open `data.html`
4. View stored user data

---

## 📌 Output

- Stores user data in browser local storage
- Displays data in table format
