# WAD Assignment - Angular User Management App

## 📌 Objective

Create an Angular application to:

- Register a user
- Login a user
- Display user data on Profile component

---

## 🛠️ Technologies Used

- Angular
- TypeScript
- HTML
- CSS
- Bootstrap (optional)

---

## ⚙️ Features

- User Registration Form
- User Login Authentication
- Store user data (Local Storage / Service)
- Profile Component to display user details
- Routing between components
